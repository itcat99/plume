# plume

Quickly build webApp

- `@plume/cli`: 命令行程序，作为发起点和项目内管理接口
- `@plume/core`: 作为不同模式\[`mode`\]下开发和打包的统一管理
- `@plume/app`: 作为react webApp项目的配置和构建 \[`app mode`\]
- `@plume/lib`: 作为组件\组件库库项目的配置和构建 \[`lib mode`\]
- `@plume/flow`: react数据流方案

## 开发前准备

安装`lerna`

```bash
npm i -g lerna
```

安装开发依赖

```bash
lerna bootstrap
```
