# plume

Quickly build webApp

- `@plume/cli`: 命令行程序，作为发起点和项目内管理接口
- `@plume/app`: 作为 react webApp 项目的配置和构建 \[`app mode`\]
- `@plume/lib`: 作为组件\组件库库项目的配置和构建 \[`lib mode`\]
- `@plume/config`: 配置文件
- `@plume/helper`: 帮助方法
- `@plume/flow`: react 数据流方案
