# plume

Quickly build webApp

- `@plume/core`: 开发框架，包含打包、路由等
- `@plume/cli`: 命令行程序
- `@plume/flow`: 数据流方案

## 开发前准备

安装`lerna`

```bash
npm i -g lerna
```

安装开发依赖

```bash
lerna bootstrap
```
